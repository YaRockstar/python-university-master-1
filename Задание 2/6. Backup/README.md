# Утилита резервного копирования директории (backup.py)

## Назначение
Утилита позволяет создавать резервные копии выбранной директории в формате ZIP-архива с сохранением структуры всех директорий.  
Все действия фиксируются в лог-файле `backup.log`.

## Требования
- Python 3.10.10
- Стандартные библиотеки (`os`, `zipfile`, `datetime`, `logging`)

## Запуск
1. Скачайте `backup.py`.
2. Запустите утилиту:
   ```bash
   python3 backup.py
